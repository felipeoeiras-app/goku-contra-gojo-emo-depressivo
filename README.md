# 🍳 OvoDourado — Landing Page

Site oficial da **OvoDourado**, especialista em pão com ovo perfeito.

## 🌐 Acessar o site

Após configurar o GitHub Pages, o site estará disponível em:
```
https://SEU-USUARIO.github.io/ovodourado/
```

---

## 🚀 Como publicar no GitHub Pages (passo a passo)

### 1. Crie um repositório no GitHub
- Acesse [github.com](https://github.com) e faça login
- Clique em **"New repository"**
- Nome sugerido: `ovodourado`
- Deixe como **Public**
- Clique em **"Create repository"**

### 2. Faça upload dos arquivos
- Dentro do repositório criado, clique em **"uploading an existing file"**
- Arraste os arquivos `index.html` e `README.md` para a área de upload
- Clique em **"Commit changes"**

### 3. Ative o GitHub Pages
- No repositório, vá em **Settings** (aba superior)
- No menu lateral, clique em **Pages**
- Em **"Branch"**, selecione `main` e pasta `/ (root)`
- Clique em **Save**

### 4. Aguarde e acesse
- Após 1-2 minutos, o site estará no ar
- A URL aparecerá na própria página de **Settings > Pages**


## ✏️ Como editar o site

Para alterar textos, cores ou seções, edite diretamente o arquivo `index.html` pelo GitHub:
- Clique no arquivo `index.html` no repositório
- Clique no ícone de lápis ✏️ (Edit this file)
- Faça as alterações e clique em **"Commit changes"**

O site atualiza automaticamente em poucos segundos.
